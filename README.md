2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:

1. Navigate to the `client` directory:

2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
